Wow! A New Linux Phone! FLX1s

Check out the FLX1s, a new contender in the Linux phone arena from Furilabs.

The FLX1s

The FLX1s is a new Linux smartphone from Furilabs, offering a fresh option for those seeking a privacy-respecting, open-source mobile device.

Get one: FLX1s on Furilabs Store

Key Features

Furi OS

The FLX1s runs Furi OS, which is based on Debian Linux. This means:

  • Full Debian package ecosystem available

  • Familiar Linux environment

  • Active community support

    • Hardware Kill Switches

    Like other privacy-focused Linux phones, the FLX1s includes hardware kill switches for enhanced security and privacy control.

    Developer Friendly

  • Python on Phone - Write and run Python scripts directly on your device

  • Full Linux Terminal - Complete command-line access

  • Open Source - FOSS (Free and Open Source Software) at its core

    • Carrier Support

    The video confirms T-Mobile compatibility, showing the phone works on major carriers.

    Alternatives Mentioned

    The FLX1s joins a growing ecosystem of Linux phones:

  • Librem 5

  • PinePhone Pro

De-Googled Experience

For those looking for an Android alternative or iPhone alternative without big tech surveillance, the FLX1s offers a de-Googled smartphone experience while still supporting some Android apps on Linux.
