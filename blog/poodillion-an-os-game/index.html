Poodillion (An OS game)

Poodillion is an operating system themed game.

What is Poodillion?

Poodillion is a game that explores operating system concepts in a playful, interactive way. It combines gaming with learning about how computers and operating systems work.

Game Concept

The game features:

  • OS-themed gameplay mechanics

  • Terminal/command-line elements

  • Educational aspects about system operations

  • Runs on Linux (including Linux phones like the Librem 5)

    • Technical Details

  • Platform: Linux, including mobile

  • Interface: Terminal/Bash-based elements

  • License: GPL (open source)

    • Try It Yourself

    The game is open source and available for you to play and explore:

  • Source Code: https://bitbucket.org/hackersgame/poodillion

  • Blog Post: https://www.hackers-game.com/2021/09/04/poodillion-an-os-game/

    • Why an OS Game?

    Games can be a great way to learn technical concepts. By gamifying operating system concepts, Poodillion makes learning about:

  • Command line operations

  • System processes

  • File systems

  • Computer architecture

More engaging and memorable than traditional tutorials.
