# Managing Linux Shell Processes

How to manage stuff you start in a shell.

This tutorial covers essential Linux shell process management, including:

- Using `bg` to send processes to the background
- Using `fg` to bring processes to the foreground
- Job control basics with the `jobs` command
- Managing locked or frozen terminal sessions
- Background and foreground process fundamentals

Understanding process management is crucial for efficient Linux terminal usage, allowing you to multitask and control running applications effectively.
