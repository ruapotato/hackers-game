# Terminal Touch Interface | Multilsh

This video demonstrates Multilsh, a touch-friendly terminal interface for Linux.

Multilsh provides an alternative way to interact with the command line using touch gestures, making terminal usage more accessible on touchscreen devices and tablets running Linux.
