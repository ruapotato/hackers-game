# Librem 5 Development Kit [Booting]

A look at the Librem 5 development kit booting up. This video shows the early stages of Purism's Linux phone development, demonstrating the boot process of the dev kit hardware that preceded the final Librem 5 device.

The Librem 5 dev kit allowed developers and enthusiasts to start working on mobile Linux applications before the phone's official release.
