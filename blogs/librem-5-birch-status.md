# Librem 5 Birch | Status

Let's see what it's like to use a Librem 5 in December 2019.

This video provides a status update on the Librem 5 Birch batch from Purism. The Birch batch was one of the early developer units shipped to backers, representing an important milestone in the development of this privacy-focused Linux smartphone.

## Birch Batch Overview

The Librem 5 was shipped in batches named after trees:
- **Aspen**: First developer batch
- **Birch**: Second batch with hardware improvements
- **Chestnut**: Third batch
- **Dogwood**: Fourth batch
- **Evergreen**: Mass production batch

## December 2019 State

This video captures the real-world experience of using a Librem 5 Birch unit as a daily device, including:

- Current software state and stability
- Hardware functionality and any issues
- Battery life observations
- Call and messaging capabilities
- App availability and usability
- Overall user experience

## Development Progress

The Birch batch represented significant progress from the initial Aspen units, with improvements to:
- Hardware design
- Thermal management
- Software stability
- Driver support

This honest assessment helps potential buyers understand what to expect from early Librem 5 hardware.
