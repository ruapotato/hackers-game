# Manjaro in a Chroot

Now you can use Manjaro in Ubuntu, OpenSuSE, Fedora, and more. This video demonstrates how to set up a chroot environment to run Manjaro within other Linux distributions, giving you access to the Arch-based package manager and software ecosystem without dual-booting or replacing your current system.
