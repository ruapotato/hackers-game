# Blog Posts

Individual markdown files for each blog post. Each file uses YAML frontmatter for metadata.

## Format

```markdown
---
id: video-slug-here
title: Video Title
date: YYYY-MM-DD
category: Category Name
tags: [tag1, tag2, tag3]
videoId: YouTubeVideoID
excerpt: Short description for listings
---

# Main Content

Blog content in markdown format...
```
