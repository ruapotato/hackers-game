# Mobile IP-native Communication

Use cellular data or Wifi at home to call or text.

This video explores IP-native communication solutions for mobile devices, demonstrating how you can use services like JMP to make calls and send texts over data connections rather than traditional cellular voice services. Whether you're on cellular data or connected to WiFi at home, IP-native communication provides flexibility and often cost savings for your mobile communication needs.
