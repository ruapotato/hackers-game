# Librem 5 Chestnut | Apps

Let's go over the Librem5 chestnut hardware changes, and talk about Librem5 apps.

This video covers the updates in the Chestnut batch of Librem 5 devices and explores the application ecosystem available for the Linux phone.

## Chestnut Hardware Changes

The Chestnut batch brought several hardware improvements over previous batches:

- Refined thermal management
- Improved component layout
- Better build quality
- Hardware bug fixes from Birch feedback

## Librem 5 App Ecosystem

The video explores what applications work well on the Librem 5:

### Native Apps
- Apps built with GTK and libhandy/libadwaita for mobile-friendly interfaces
- GNOME applications adapted for small screens
- Purpose-built Linux phone applications

### App Sources
- PureOS software repositories
- Flatpak applications
- Apps from the broader Linux ecosystem
